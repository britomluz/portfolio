¿Qué significa ser diseñadora de código?

Soy  

Soy desarrolladora fullstack java con una ávida pasión por todo lo relacionado con el diseño y la creatividad. También soy diseñadora gráfica enfocada en brindar soluciones a través de aplicaciones web intuitivas con flujo de contenido reflexivo.

Mi fuerte es combinar la mejor parte de la productividad, del proyecto de identidad de la marca y el desarrollo de tecnologías para crear aplicaciones que brinden crecimiento a mis clientes.

Como programadora y diseñadora me enfoco especialmente en la usabilidad y el diseño y tengo un amor especial por los lenguajes de programación back-end Me esfuerzo por ofrecer soluciones profesionales y funcionales a los clientes.

Mi portfolio es variado y contiene mis proyectos de desarrollo realizados mientras estudiaba como también algunos de mis trabajos de diseño de marca.

    Aplicaciones web

    Acquerello Italian Food

    Restaurant App

    Esta aplicación web facilita el manejo de comprar, reservar mesa and manejar el estado de los pedidos del usuario. El usuario puede descargar la factura del pedido y recibir por mail el estado de su pedido. Los administradores pueden manejar el stock, agregar productos o categorías y manejar los pedidos y las reservas. Construida con su propia API Rest , Java, Spring, Spring boot, Spring Security, Gradle, HTML5, CSS3, Bootstrap 5 y VueJs.

    Repo Github View site

    MBB Bank

    Homebanking App

    MBB Bank es una aplicación bancaria que permite al usuario crear su cuenta bancaria, realizar transacciones, solicitar préstamos, y crear tarjetas de crédito o débito. El usuario puede ver su extracto de cuenta, un listado de transacciones con sus detalles para descargar su comprobante, y una vista del resumen de tu tarjeta de crédito/débito existente. Construida con su propia API Rest , Java, Spring, Spring boot, Spring Security, Gradle, HTML5, CSS3, Bootstrap 5 y VueJs.

    Repo Github View site
